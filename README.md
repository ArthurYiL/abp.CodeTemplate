简介

用abp 框架快两年了，用它完成了多个项目，作为CTO同时也作为架构师，在应对中小型项目时候，我们通常选择ABP（内部大型的物联网架构采用自己的框架），感觉这款框架真心不错。虽然开源社区有也有很多写了几套代码生成器，但是我用完之后，总是感觉不能达到我自己想要的效果，我个人还是比较喜欢一步到位，批量生成，所以就写了这套基于codesmith的代码生成器，这一套在项目中还算稳定。
